# Vision

打造一个编辑式极简风格的个人博客（petermu.com）和阅读日志（reading.petermu.com），用暖色奶油底 + 深绿点缀的设计语言，以 Obsidian 为内容源、Readwise 为阅读数据源。

1. 视觉上达到 jiyongshi.com 的审美水平
2. 极简架构：无数据库、无认证、无评论
3. 内容工作流：Obsidian 写作 → git push → 自动部署
4. 阅读追踪：通过 Readwise API 展示阅读高亮和笔记
