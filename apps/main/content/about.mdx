# About

我是 Peter，一个在医学院做科研的人，同时也是一个停不下来的学习者。

白天的工作是科研和教学，晚上的时间属于我自己——写代码、读书、复盘、做各种实验。这个网站就是我的"实验室"，记录我在 AI、编程、阅读、个人成长这些方向上的探索。

## 为什么叫 Lab？

因为我做的每件事都像是实验。学 AI 编程是实验，用 Readwise 搭建阅读系统是实验，写复盘文章是实验，甚至这个博客本身也是实验——它是用 Claude Code 从零写出来的。

我不是专家，也不追求做专家。我更像是一个"什么都想试试"的人。有些实验成功了（比如用 AI 在 30 分钟内完成过去需要 3 小时的备课），有些还在进行中（比如学机器学习）。

## 我相信什么

- **边学边做**比"学完再做"有效得多
- **完成比完美重要**——先做出来，再迭代
- **接受自己的节奏**——我是冲刺型选手，不是马拉松型
- **创造比消费更有意义**——从知识的消费者变成创造者
