## 📚 常用命令

```bash
# 开发环境
npm run dev              # 启动开发服务器
npm run build            # 构建生产版本
npm run start            # 启动生产服务器

# 测试
npm run test             # 运行测试
npm run test:coverage    # 测试覆盖率
npm run test:api         # API 集成测试
npm run test:notion      # Notion 连接测试
npm run test:supabase    # Supabase 连接测试

# 代码质量
npm run lint             # 代码检查
npm run analyze          # Bundle 分析
```

## 🔑 重要文件路径

```
项目结构：
├── app/                  # Next.js 应用代码
├── components/           # React 组件
├── lib/                  # 工具库
│   ├── notion.ts        # Notion API 集成
│   └── supabase/        # Supabase 客户端
├── scripts/             # 脚本文件
│   ├── supabase-init.sql    # 数据库初始化
│   └── test-*.js            # 测试脚本
├── docs/                # 文档中心
└── .env.example         # 环境变量示例
```
