## 📝 发布文章

### 第一步：创建新文章

1. 打开 **文章数据库**
2. 点击右上角的 **"New"** 按钮
3. 填写以下必填信息：
   - **标题**（Title）：文章标题
   - **作者**（Author）：您的名字
   - **日期**（Date）：发布日期
   - **分类**（Category）：选择一个分类
     - Technology（技术）
     - Design（设计）
     - Productivity（效率）
     - Life（生活）
   - **标签**（Tags）：添加相关标签（可多选）

### 第二步：编写内容

在 Notion 页面中直接编写，支持：

- **Markdown 语法**：标题、列表、引用、代码块等
- **富文本编辑**：加粗、斜体、下划线
- **插入元素**：
  - 图片（建议宽度 800px 以上）
  - 代码块（支持语法高亮）
  - 引用块
  - 分割线

### 第三步：优化文章

#### 添加封面图

- 点击页面顶部 "Add cover"
- 上传高质量图片（建议 1200x630px）
- 或使用 Unsplash 免费图库

#### 设置摘要

在文章开头写一段 100-200 字的摘要，这将显示在文章列表中。

#### 添加目录

对于长文章，使用多级标题自动生成目录：

```
# 一级标题
## 二级标题
### 三级标题
```

### 第四步：发布文章

1. 检查所有内容无误
2. 确保必填字段都已填写
3. 保存 Notion 页面
4. 等待 5-10 分钟，网站缓存更新
5. 访问您的博客查看新文章

> ⚡ **快速发布技巧**：使用 Notion 模板功能，创建文章模板，包含常用的结构和格式。

## 🚀 管理项目作品

### 添加新项目

1. 打开 **项目数据库**
2. 创建新页面，填写：
   - **项目名称**
   - **项目描述**（简短介绍）
   - **详细说明**（在页面内容中）
   - **技术栈**（Technologies）：使用的技术
   - **项目链接**：
     - Demo URL（演示地址）
     - GitHub URL（源码地址）
   - **项目类型**：
     - 网站应用
     - 开源项目
     - 设计作品
     - 其他

### 项目展示技巧

1. **添加项目截图**
   - 首页截图
   - 核心功能展示
   - 移动端效果

2. **编写项目故事**
   - 项目背景
   - 解决的问题
   - 技术亮点
   - 未来计划

3. **展示代码片段**
   ```javascript
   // 展示核心代码逻辑
   function amazingFeature() {
     // 您的代码
   }
   ```

## 📚 更新书架

### 添加新书

1. 打开 **书籍数据库**
2. 创建新页面，填写：
   - **书名**（必填）
   - **作者**（必填）
   - **阅读状态**：
     - 在读（reading）
     - 已读（completed）
     - 想读（want-to-read）
   - **评分**：1-5 星
   - **分类**：技术、设计、人文等
   - **出版信息**：
     - 出版年份
     - 页数
     - ISBN

### 撰写读书笔记

在书籍页面内容区域：

1. **内容摘要**：书籍核心内容
2. **精彩语录**：值得记住的句子
3. **个人感悟**：您的思考和收获
4. **实践应用**：如何应用书中知识

### 书架管理技巧

- 定期更新阅读进度
- 为喜欢的书写详细书评
- 使用标签分类（如：必读、推荐、专业书籍）

2. 填写工具信息：
   - **工具名称**
   - **简短描述**

1. **使用场景**：什么情况下使用
2. **核心功能**：主要特点

4. **使用技巧**：提高效率的方法
5. **替代方案**：类似工具对比

## 📊 查看网站数据

### Google Analytics

1. 访问 [Google Analytics](https://analytics.google.com)
2. 查看关键指标：
   - **访问量**：每日/每周/每月
   - **热门文章**：最受欢迎的内容
   - **访客来源**：搜索引擎、社交媒体、直接访问
   - **设备分布**：桌面端 vs 移动端

### 内容表现分析

定期（建议每月）分析：

1. **最受欢迎的文章类型**
2. **读者停留时间**
3. **跳出率**
4. **转化率**（订阅、评论等）

根据数据调整内容策略。

## ❓ 常见问题

### Q: 发布文章后多久能在网站上看到？

A: 通常 5-10 分钟内。如果超过 30 分钟还看不到，可以尝试：

1. 检查 Notion 中必填字段是否完整
2. 在 Vercel 后台手动触发重新部署

### Q: 如何修改已发布的文章？

A: 直接在 Notion 中编辑对应文章，保存后等待缓存更新即可。

### Q: 可以定时发布文章吗？

A: Notion 不支持定时发布，但您可以：

1. 提前写好文章，设置未来日期
2. 到发布日期时，将文章状态改为"已发布"

### Q: 如何处理图片？

A: 建议：

1. 图片宽度至少 800px
2. 使用 WebP 或优化后的 JPG 格式
3. 为图片添加 alt 描述
4. 使用 CDN 服务（如 Cloudinary）托管大图

### Q: 如何提高文章 SEO？

A:

1. 使用描述性的标题
2. 在文章开头包含关键词
3. 使用合适的标题层级
4. 添加相关标签
5. 保持内容原创和高质量
