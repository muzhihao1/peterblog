# 博主快速参考卡

> 🚀 常用操作的快速指南，建议打印或收藏此页面

## 📝 发布新文章

1. 打开 Notion 文章数据库
2. 点击 "New" 创建新页面
3. 填写必填项：
   ```
   ✓ Title: 文章标题
   ✓ Author: 您的名字
   ✓ Date: 今天日期
   ✓ Category: 选择分类
   ✓ Tags: 添加标签
   ```
4. 在页面中编写内容
5. 保存后等待 5-10 分钟即可看到

## 🔄 更新已有内容

### 修改文章

1. 在 Notion 中找到对应文章
2. 直接编辑内容
3. 保存即可（自动更新）

### 添加新项目

1. 打开项目数据库
2. 填写项目信息和截图
3. 添加 Demo/GitHub 链接

### 更新书架

1. 打开书籍数据库
2. 更改阅读状态
3. 添加读书笔记

### 推荐新工具

1. 打开工具数据库
2. 填写工具信息
3. 写详细评测

## 🔧 常见问题快速解决

### 文章不显示？

- 检查 Published 是否勾选
- 检查必填字段是否完整
- 等待 10 分钟缓存更新

### 图片不显示？

- 使用外部图床
- 图片宽度 > 800px
- 使用 HTTPS 链接

### 更新不生效？

- 清除浏览器缓存
- 手动触发 Vercel 重新部署
