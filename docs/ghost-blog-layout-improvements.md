# Ghost Blog 排版改进方案

基于 Ghost Blog 的设计理念，这里是详细的改进建议，帮助你的博客实现更优雅、更专注于内容的阅读体验。

## 核心改进要点

### 1. **字体与排版**

- **基础字体大小**: 从 16px 提升到 21px
- **行高**: 从 1.5 提升到 1.7
- **段落间距**: 增加到 1.8rem
- **字体选择**: 优先使用系统字体，确保中文显示效果最佳

### 2. **内容宽度**

- **最大宽度**: 限制在 740px，提供最佳阅读体验
- **移除侧边栏**: 专注于文章内容，避免干扰
- **居中对齐**: 内容始终保持居中，创造沉浸式阅读体验

### 3. **视觉层级**

- **标题尺寸**:
  - H1: 48px → 54px (文章标题)
  - H2: 36px
  - H3: 28px
- **字重调整**: 使用更大胆的字重(700-800)突出标题
- **间距优化**: 标题前后留出充足空白

### 4. **简化设计元素**

- **移除不必要的装饰**: 减少阴影、边框等视觉噪音
- **简化交互元素**: 只保留必要的功能
- **清理元数据**: 精简显示信息，突出发布时间和作者

### 5. **色彩方案**

- **主色调**: 深灰色文字(#1a1a1a)在白色背景上
- **次要文字**: 使用 #5a5a5a 表示次要信息
- **链接样式**: 下划线链接，悬浮时变色
- **减少色彩**: 整体使用黑白灰，营造专业感

## 实施步骤

### 第一步：更新全局样式

```css
/* 在 globals.css 中添加 */
@import "./styles/ghost-inspired.css";

/* 覆盖现有的 prose 样式 */
.prose-blog {
  @apply ghost-article;
}
```

### 第二步：简化文章页面布局

1. 移除侧边栏目录
2. 简化顶部导航
3. 精简文章元数据显示
4. 移除社交分享按钮和反应组件

### 第三步：优化移动端体验

- 移动端字体大小: 18px
- 减少内边距
- 确保触摸目标足够大

### 第四步：性能优化

- 移除不必要的动画
- 减少 JavaScript 交互
- 优化图片加载

## 对比效果

### 改进前

- 复杂的布局结构
- 多种交互元素
- 较小的字体
- 信息密度高

### 改进后

- 简洁的单栏布局
- 专注于内容
- 更大更易读的字体
- 充足的留白

## 具体修改建议

### 1. 修改文章页面组件

```tsx
// 使用新的 Ghost 风格页面
// 将 app/posts/[slug]/page.tsx 的内容替换为 ghost-style-page.tsx 的内容
```

### 2. 更新 CSS 变量

```css
:root {
  --article-font-size: 21px;
  --article-line-height: 1.7;
  --content-max-width: 740px;
  --heading-font-weight: 700;
}
```

### 3. 移除多余组件

- TableOfContents (目录)
- ArticleReactions (文章反应)
- SocialShare (社交分享)
- 复杂的标签显示

### 4. 简化导航

```tsx
// 固定简洁的顶部导航
<nav className="fixed top-0 bg-white/80 backdrop-blur">
  <Link href="/">博客名称</Link>
  <span>阅读时间</span>
</nav>
```

## 可选增强功能

### 1. 平滑滚动进度条

- 极细的顶部进度条
- 不干扰阅读

### 2. 暗色模式

- 保持同样的简洁设计
- 使用深色背景和浅色文字

### 3. 代码高亮

- 使用简洁的单色主题
- 确保对比度足够

## 测试建议

1. **阅读测试**: 找几个人实际阅读文章，收集反馈
2. **性能测试**: 确保页面加载速度快
3. **响应式测试**: 在不同设备上测试
4. **可访问性测试**: 确保所有人都能舒适阅读

## 总结

Ghost Blog 的设计哲学是"内容为王"。通过这些改进，你的博客将：

- 提供更好的阅读体验
- 减少视觉干扰
- 提高内容的专注度
- 创造更专业的印象

记住：好的设计是看不见的设计。让读者专注于你的文字，而不是界面。
