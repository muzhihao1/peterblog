# 图片存储快速入门指南

## 🚀 快速开始

### 方案一：GitHub + jsDelivr（推荐）

#### 1. 初始设置
```bash
# 创建 blog-assets 仓库（在 GitHub 上创建公开仓库）
# 克隆到本地
git clone https://github.com/YOUR_USERNAME/blog-assets.git
```

#### 2. 上传单张图片
```bash
# 交互式上传
node scripts/upload-image.js

# 或命令行模式
node scripts/upload-image.js ./my-image.jpg projects personal-blog
```

#### 3. 批量处理项目图片
```bash
# 安装依赖
npm install sharp commander

# 运行批量处理脚本
npx tsx scripts/batch-upload-project-images.ts
```

### 方案二：Cloudinary（备用）

#### 1. 注册并配置
1. 访问 [Cloudinary](https://cloudinary.com) 注册账号
2. 获取 API 凭证
3. 添加到 `.env.local`：
```env
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```

#### 2. 上传图片
```bash
# 安装 Cloudinary SDK
npm install cloudinary

# 上传本地图片
npx tsx scripts/cloudinary-upload.ts ./image.jpg project-slug

# 从 URL 上传
npx tsx scripts/cloudinary-upload.ts --url https://example.com/image.jpg project-slug
```

## 📸 图片优化工具

### 优化单张图片
```bash
# 生成缩略图（800x600）
npx tsx scripts/optimize-images.ts single ./input.jpg ./output.jpg --preset thumbnail

# 生成截图（1200x800）
npx tsx scripts/optimize-images.ts single ./input.jpg ./output.jpg --preset screenshot
```

### 批量优化
```bash
# 优化整个目录
npx tsx scripts/optimize-images.ts batch ./images-folder --preset thumbnail
```

### 为项目生成所有格式
```bash
# 自动生成缩略图和截图
npx tsx scripts/optimize-images.ts project my-project-name --input ./original.jpg
```

## 📝 在 Notion 中使用

### 添加图片到项目
1. 打开 Notion 项目数据库
2. 在 `Thumbnail` 字段选择 "Embed link"
3. 粘贴 CDN 链接：
   - GitHub CDN: `https://cdn.jsdelivr.net/gh/username/blog-assets@main/projects/slug/thumbnail.jpg`
   - Cloudinary: `https://res.cloudinary.com/your-cloud/image/upload/q_auto,f_auto/blog/projects/slug/thumbnail`

## 🎯 最佳实践

### 图片规格
- **项目缩略图**: 800x600px, < 200KB
- **项目截图**: 1200x800px, < 500KB
- **博客封面**: 1200x630px（社交分享优化）
- **格式**: JPEG（质量 85-90%）或 WebP

### 命名规范
- 使用小写字母和连字符
- 缩略图: `thumbnail.jpg`
- 截图: `screenshot-1.jpg`, `screenshot-2.jpg`
- 避免中文和特殊字符

## 🛠️ 常见问题

### Q: jsDelivr 链接无法访问？
1. 确保仓库是公开的
2. 检查文件路径是否正确
3. 等待 CDN 缓存更新（约 10 分钟）

### Q: 图片加载太慢？
1. 使用图片优化工具压缩
2. 考虑使用 Cloudinary 的自动优化
3. 检查图片尺寸是否过大

### Q: 如何批量迁移现有图片？
```bash
# 使用批量上传脚本
npx tsx scripts/batch-upload-project-images.ts

# 选择自动模式，使用默认图片
# 或选择交互模式，手动指定每个项目的图片
```
