# 个性化设置功能说明

## 功能介绍

个性化设置功能允许用户根据自己的喜好和需求定制博客的视觉体验。通过点击导航栏右侧的设置图标（⚙️）即可打开设置面板。

## 主要功能模块

### 1. 主题设置

- **预设主题**：提供6种精心设计的主题
  - 默认主题（Default）
  - 暗色主题（Dark）
  - 护眼主题（Eye Care）
  - 高对比主题（High Contrast）
  - 紫色主题（Purple）
  - 蓝色主题（Blue）
- **主题模式**：
  - 亮色模式
  - 暗色模式
  - 系统跟随（根据操作系统设置自动切换）

### 2. 字体设置

- **字体大小**：
  - 小号（14px）
  - 中号（16px）- 默认
  - 大号（18px）
  - 特大号（20px）
- **字体系列**：
  - 系统默认
  - 思源宋体（适合长文阅读）
  - 思源黑体（现代简洁）
  - 霞鹜文楷（文艺风格）

### 3. 布局设置

- **布局密度**：
  - 紧凑（适合快速浏览）
  - 舒适（默认，平衡阅读体验）
  - 宽松（适合专注阅读）
- **圆角风格**：开启/关闭界面元素的圆角效果

### 4. 辅助功能

- **减少动画**：为对动画敏感的用户减少或禁用界面动画
- **高对比度**：增强文字和背景的对比度，提高可读性

## 使用说明

1. 点击导航栏右侧的设置图标 ⚙️
2. 在弹出的侧边栏中调整各项设置
3. 所有更改会实时生效并自动保存
4. 设置会保存在浏览器本地，下次访问时自动应用

## 技术实现

- 使用 React Context 管理主题状态
- 通过 CSS 变量实现主题切换
- 使用 localStorage 持久化用户偏好
- 支持系统主题自动检测（prefers-color-scheme）
